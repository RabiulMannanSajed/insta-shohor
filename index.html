1. Session storage vs Local storage

Session storage:when we store data in session storage that mean the data only passes only for the particular session when the user closes the window that data is lost session stories data not being sent to the server.it stores a minimum of 5 MB of data.
Local storage:local storage is as same as session storage the advantage is here is that doesn't clear itself and the user closes is using the tab this data is stored in his memory when again the user comes to the same website that data is still stored in this website local storage. it has the highest memory capacity this depends on your device and Browser. these store API follows the same-origin policy
